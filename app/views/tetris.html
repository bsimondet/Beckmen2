Try dragging the blocks into the spots to change the number of columns, rows, colors and drop speed. Remember to put numbers in the columns, rows, and drop speed sections for them to work correctly!

Remember that changing the number of rows/columns can make some columns in space the window doesn't show! You can also restrict the area within the window that can be used for Tetris

After dragging in a color block, it can take a few seconds to take effect.

Higher drop speeds will result in the game slowing down, and lower drop speeds will result in the game moving more quickly!
